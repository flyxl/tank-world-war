# 坦克世界大战

[English](README.md)

基于 Babylon.js 的 3D 坦克对战游戏，支持 PC 和移动端。

## 功能特性

- **多种坦克**：轻型、中型、重型、歼击车，各有不同属性
- **4 张战斗地图**：沙漠战场、城市废墟、森林草原、雪地冻原
- **AI 对战**：有限状态机驱动的敌方坦克（巡逻→追击→攻击→撤退）
- **拾取系统**：战场散落生命、火力、速度、护盾增益，击毁敌人有概率掉落
- **升级系统**：使用金币升级护甲、火力、速度、装填速度
- **跨平台**：PC 键鼠操控，移动端虚拟摇杆 + 强制横屏
- **视觉效果**：PBR 材质、后处理（Bloom/SSAO/FXAA）、粒子效果、天气系统

## 技术栈

| 技术 | 用途 |
|------|------|
| Babylon.js 9.x | 3D 渲染引擎 |
| TypeScript | 主开发语言 |
| Vite | 构建与开发服务器 |

## 快速开始

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建生产版本
pnpm build

# 预览构建产物
pnpm preview
```

## 操控方式

### PC

| 操作 | 按键 |
|------|------|
| 移动 | WASD / 方向键 |
| 瞄准 | 鼠标移动 |
| 开火 | 鼠标左键 / 空格 |
| 缩放 | 鼠标滚轮 |

### 移动端

| 操作 | 控制 |
|------|------|
| 移动 | 左侧摇杆 |
| 瞄准 | 右侧摇杆 |
| 开火 | 开火按钮 |

## 部署

构建后的 `dist/` 目录为纯静态文件，可部署至任何平台：

- **Vercel**：`npx vercel`
- **Netlify**：拖拽 `dist/` 上传
- **Nginx**：将 `dist/` 配置为网站根目录
- **Docker**：使用 `nginx:alpine` 镜像挂载 `dist/`

## 项目结构

```
src/
├── core/           # 核心系统（输入、相机、音频）
├── entities/       # 游戏实体（坦克、炮弹、拾取物）
├── systems/        # 游戏系统（AI、战斗、粒子、升级）
├── ui/             # 界面（主菜单、HUD、车库、战斗结果）
├── world/          # 世界（地形、环境、天空、天气）
├── utils/          # 工具函数
├── Game.ts         # 游戏主控制器
└── main.ts         # 入口文件
```
